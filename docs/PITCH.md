**[S6] 0:50–1:00 — デモ(画面が動いています)**

**[S7] 1:00–1:13 — ビジネス① 商流の逆転(キメ2)**

## スライド自体が動きます

- **S2/S3/S7/S8**: 大きな数字はスライド表示と同時に**カウントアップ**(約1.3秒)。回り切ってから読み上げるとキマる
- **S3**: 右下に**240万→102万→30万人の減少カーブが自動描画**(約2秒)。「作る人が消えていく」と言いながら指す
- **S6**: 静止画ではなく**アプリと同じライブシーンが実駆動** — 雑草検出ボックスが2秒おきに出現、NDVIドローンが巡回。表示後1〜2秒「見せる間」を取る
- **S7**: 課題の数字が回ったあと、逆転した商流が左から立ち上がる — 「参加者が先に買う」ノードから指でなぞる
- **S8**: 最後に締めのフレーズ「お金は都市から地方へ…」が浮かび上がる(表示から約1.2秒後)— このフレーズは**スライドと同時に声を重ねる**
- 全スライドで見出し・カードが順に立ち上がる — 表示した瞬間に話し始めてOK

## 想定Q&Aの弾

- **なぜあなたたちがやるの?** → 空(運航管理・地方送客)×土地(遊休地開発・施設建築)×デジタル(社会インフラSI・農業ICT)。ポート構想は3産業の交点にあり、単品アグリテックには出せない総合力。S12参照
- **規制は?** → 野菜リターン=前払購入契約で金商法対象外。金銭化は二種業提携の2トラック設計。農地は所有せずリース(2009年改正・最長50年)
- **ロボットは高い?** → 小口参加者が1台を共同稼働させるのが本質。RaaS(inaho 15%/AGRIST 10%)なら初期投資ゼロ、中山間は15〜100万円級から
- **既存のシェア畑・楽天Ragriとの違い?** → Ragriは人手依存のまま2020年終了。「観察型オーナー制×フィジカルAI無人作業」の掛け算は空白地帯
- **「売れてから作る」は本当に成立する?** → 国も契約取引を98万t→145万tへ拡大方針(加工・業務用)。CrowdFarmingが樹木30万件・€65Mで先行実証。Agri Portはそれを「観察できる」形で個人に開く
- **儲かるの?** → 中抜き30pt(48.5%→71.8〜78.5%、農水省調査)が原資+データ・カーボン・ツーリズムの4層収益
- **食料安保にどこまで効く?** → 再生可能9.4万haは眠る供給力。輸入依存62%の緩衝材であり、「作れる国」を維持する装置
- **その先の構想は?(チームレーダー — LPのBEYOND 2040にも掲載)** →
  ①**農業版Airbnb=身体のシェア**: 世界中のフィジカルAIの身体をスマホから5分借りる。テレオペの操作ログは艦隊の模倣学習データに還流(アプリに構想デモの予約ボタンあり)
  ②**アリ農業=スワーム**: 大型1台より手のひら1万台。冗長性・24時間・株単位観察はdMRVの前提条件(LPのフィジカルAI体系に設計思想として掲載)
  ③**回遊農業**: 栽培コンテナが天候を追ってポート間を自律移動 — ポートは文字通り「港」、ユニットは「船」になる
  ④**惑星スケール農地**: 生命のいない環境を丸ごと農地に — 月面温室はその一歩目
