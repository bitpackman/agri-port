## アプリ実演に切り替える場合(S4で)

1. スライド4を表示したまま:「実物をお見せします」
2. 別タブの **[app.html](https://agri-port.pages.dev/app.html)** へ(事前に開いてタブを温めておく)
3. 見せる順(約20秒): ライブ観察の**ロボット視点**タブ(検出ボックスが出る)→ **リターンセンター**で 野菜/お金/地球 をワンタップ切替 → 戻ってS5へ
4. 作業ログは5秒ごとに自動更新されるので、話している間も画面が「生きて」見えます

## 想定Q&Aの弾

- **規制は大丈夫?** → 野菜リターンは前払購入契約で金商法対象外(購入型CFと同じ整理)。金銭化する段階で第二種金融商品取引業者と提携する2トラック設計。農地は所有せずリース(2009年改正で自由化・最長50年)
- **ロボットは高いのでは?** → 1台1,000万円超でも、多数の小口参加者が1台を共同稼働させるのがこのスキームの本質。しかもinaho(収穫量×15%)やAGRIST(出荷額10%)のRaaSなら初期投資なしで従量課金。中山間なら獣害ロボ+草刈りの15〜100万円級から始められる
- **既存のシェア畑や楽天Ragriと何が違う?** → Ragriは人手依存のまま2020年に終了。「観察型オーナー制×フィジカルAI無人作業」の掛け算は国内外に先行事例がない空白地帯
- **本当に儲かるの?** → 中抜きされていた流通マージン約30ポイント(手取り48.5%→直販71.8〜78.5%、農水省調査)が原資。加えて生育データ・カーボンクレジット・体験ツーリズムの複層収益
- **なぜ日本から?** → 高齢化最先端(65歳以上69.5%)=世界で最初に「人間なし農業」を強制される市場だから。解けば韓国(51%)→EU(55歳+61%)→中国へ順に輸出できる
