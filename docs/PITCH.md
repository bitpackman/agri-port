## 想定Q&Aの弾

- **なぜあなたたちがやるの?** → 空(運航管理・地方送客)×土地(遊休地開発・施設建築)×デジタル(社会インフラSI・農業ICT)。ポート構想は3産業の交点にあり、既存アグリテック(ロボット単品/アプリ単品)には出せない総合力。S11参照
- **規制は?** → 野菜リターン=前払購入契約で金商法対象外。金銭化は二種業提携の2トラック設計。農地は所有せずリース(2009年改正・最長50年)
- **ロボットは高い?** → 小口参加者が1台を共同稼働させるのが本質。RaaS(inaho 15%/AGRIST 10%)なら初期投資ゼロ、中山間は15〜100万円級から
- **既存のシェア畑・楽天Ragriとの違い?** → Ragriは人手依存のまま2020年終了。「観察型オーナー制×フィジカルAI無人作業」の掛け算は空白地帯
- **儲かるの?** → 中抜きされていた流通マージン約30pt(手取り48.5%→71.8〜78.5%、農水省調査)が原資+データ・カーボン・ツーリズムの複層収益
- **食料安保にどこまで効く?** → 再生可能9.4万haは水田換算で概ね数十万トン級の生産余力。まず「眠る供給力を市場に戻す装置」であり、輸入依存62%の緩衝材になる
