## 1. なぜ解くべき課題なのか

食は衣食住で唯一、1日も止められない生存インフラである。その作り手が日本では25年で57%減り(240万→102万人)、70歳以上が55%。農水省自身が20年後30万人と試算する「人口動態が確定させた崩壊」が進む。現在の方法——新規就農の促進——では解決できない。若手の流入は年1.6万人で、流出6.8万人の4分の1に過ぎないからだ。人を増やす政策では埋まらない。「人がそこに住んで働く」前提の産業設計そのものを変える必要がある。

**(現在の方法ではなぜ解決できないのか)** 補助金・移住促進・機械化支援はすべて「個人が現地で担う」前提の対症療法である。関わる方法が移住しかなく都市の関心と資本に入口がない/労働が土地に縛られ、その身体が70歳を超えた/1軒では1,000万円超の機械に投資できない——この3つの構造が根本原因であり、構造ごと組み替えない限り、流出は止まらない。

## 2. なぜ心が動くか

スマホを開くと、新潟の棚田で除草ロボが雑草をAI検出して抜き、ドローンが生育をスキャンし、選果場でヒューマノイドが自分の区画のトマトを箱に詰めている——1口1万円のオーナーとして、その全部をライブで観察できる。72歳の田中さんの田んぼは荒れ地にならず、彼は艦隊を見回るポートパートナーになり、秋には谷に収穫客が戻る。農業が「移住」から「ログイン」に変わり、誰もが地球の食に参加している未来。私たちはこの風景を見たい。デモは今日、動いている。

## 3. なぜ現実世界に与えるインパクトが大きいか

対象は一つの農場ではなく産業構造そのものだ。起点は「作付け前に需要と価格が確定する」逆転で、そこからドミノが倒れる——農家の手取りが48.5%から最大78.5%へ(流通構造の変化)/豊作暴落と産地廃棄が設計上消える(価格形成の変化)/数量確定で計画配送が可能になり物流2024年問題に適合(物流の変化)/農業が1口1万円の投資適格資産になる(金融の変化)/艦隊の観察データがカーボン検証台帳を兼ねる(環境市場の変化)。国内の再生可能な荒廃農地9.4万haの先に、世界の耕作放棄地約1億haと飢餓6.7億人という地球規模の課題が同じ構造で待っている。

## 4. なぜサステナブルになっているか

補助金ではなく、市場流通で中抜きされていた約30ポイントの流通マージンを原資とする事業である。払い続けるのは参加者——出資=前払購入なので「安全で顔の見える食材の産直」として対価に見合い続け、リターンは野菜・金銭・カーボン証明書の3択で多様な動機を取り込む。農家は地代と手取り増、地域は雇用と関係人口、参加者は食と体験を得る三方よし。法的にも野菜リターンは金商法の外で今日から開始でき、スケール時に投資型へ拡張する二段設計。さらに艦隊は走るほどデータが貯まり賢くなる(フリートラーニング)ため、規模拡大とともに限界費用が下がり続ける。
