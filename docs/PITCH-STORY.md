## 物語の背骨

```
田中さん(課題の顔)→ 数字 → なぜ止められなかったか(構造)
 → フィジカルAIが労働と土地を切り離す(活路)
 → では誰が・何のために参加するのか(動機の3層: 愛着・当事者性・希少性)★
 → 観る(デモ)→ 決める(台風の選択)→ 操る(リビングから耕す)★
 → 参加のお金が仕組みを回す(売れてから作る+遊んだ分だけ国土が再稼働)
 → 構造比較(AS-IS/TO-BE)
 → 田中さんのその後 → 世界中の田中さん
 → あなたの参加が、飢餓を減らす(クライマックス)★
 → チーム → クロージング
```

全16枚: `1 タイトル / 2 田中さん / 3 数字 / 4 なぜ未解決 / 5 活路 / 6 参加の動機 / 7 ライブデモ(観る) / 8 決める / 9 操る / 10 ビジネス / 11 構造AS-IS,TO-BE / 12 田中さんのその後 / 13 世界中の田中さん / 14 HOPE(飢餓) / 15 チーム / 16 クロージング`

> ⚠️ 田中さんは統計から組み立てた架空のモデルケース(仮名)。スライドに明記済み。チームに実の原体験があればS2を実話に差し替えるのが最強。

## 90秒版(10枚: 1→2→3→4→5→6→8→10→14→16)

> ライブデモ(S7)・操る(S9)・構造(S11)・その後(S12)・世界(S13)・チーム(S15)を飛ばす。デモはクロージングのQRに託し、チームはクロージングで一言。

**[S1] 0:00–0:07 — タイトル**
「Agri Port、アプリDE農業です。まず、ひとりの話をさせてください。」

**[S2] 0:07–0:20 — 田中さん**
「田中さん、72歳。新潟の谷で米を50年、いまはこの谷で最後のひとりです。膝は限界。**彼が手を止めた日、この谷の田んぼはまとめて『荒廃農地』という統計に変わります。**(半拍)田中さんは仮名です。でも、数字は全部本物です。」

**[S3] 0:20–0:31 — 数字**
「70歳以上が55%。担い手は25年で57%減。作っても手取りは半分以下で、豊作の年は畑に捨てる。**作る人が消えていく。作っても、報われない。**」

**[S4] 0:31–0:42 — なぜ止められなかったのか**
「誰も怠けていません。**産業のつくりの問題**です。関わる方法が移住しかない。労働が土地に縛られている。1軒では1,000万円の機械に投資できない。」

**[S5] 0:42–0:52 — 活路(キメ1)**
「フィジカルAIは、この構造を壊せます。**労働と土地を切り離せる**んです。ロボットが現地の身体になり、人はどこからでも繋がれる。だから私たちは、農業を『移住』から『ログイン』に変えます。」

**[S6] 0:52–1:04 — 参加の動機(新しい心臓部)**
「では、誰が何のために参加するのか。人は、**経験できないものにいちばん心を動かされます**。農業をやりたい人は少ない — でも食の出どころに関わりたい人は膨大にいる。だから参加は3層です。毎日**観る**愛着。監督として**決める**当事者性。そして誰も買えなかった体験を**操る**希少性。お金を出して終わりのクラファンとは、構造が違います。」

**[S8] 1:04–1:14 — 決める**
「例えば — 『台風6号接近中。早期収穫か、粘って糖度を取るか』。AIがリスクを出し、**決めるのはあなた**。収穫は、あなたの采配の記録になります。」

**[S10] 1:14–1:24 — ビジネス(キメ2)**
「参加のお金は『売れてから作る』を実現します。作付け前に完売、手取りは48.5%から最大78.5%へ、廃棄は設計上ゼロ。**遊んだ分だけ、国土が再稼働する**んです。」

**[S14] 1:24–1:33 — HOPE(キメ3・クライマックス)**
「そしてこの物語の終着点です。飢餓6.73億人。でも世界には眠る耕作放棄地が1億haある。あなたの1万円が、あなたの5分の操縦が、眠る土地を畑に戻していく — **あなたの参加が、飢餓を減らす。**」

**[S16] 1:33–1:40 — クロージング**
「私たちは空・土地・デジタルの現場から集まった3人。地球で無人農業を解けない種は、火星でも耕せない — まず、田中さんの谷から。デモはこのQRから、いま触れます。」

## 演出メモ

- **キメは3つ**: S5「移住からログインへ」/ S10「遊んだ分だけ、国土が再稼働する」/ **S14「あなたの参加が、飢餓を減らす」(最大)**
- S2は静かに。数字もカタカナ語もなし — 会場の空気をここで変える
- S6が新しい心臓部: 「クラファンとは構造が違います」をはっきり言う(メンター指摘への公開回答)
- S8(台風の選択)は会場に「皆さんならどっちですか?」と振ってよい
- S14は画(市場を駆ける子どもたち)に語らせる — 「あなたの1万円が」からはゆっくり、1語ずつ
- S16のQRは3秒黙って見せる

## Q&Aの弾

`docs/QA.md`(35問)を使用。この構成で特に飛びやすいのは:
- 「クラファンと何が違う?」→ S6で先回りして答えているが、深掘りされたら「財布だけの参加 vs シーズン中ずっと判断と体験で関わる参加」(QA H-1/H-2)
- 「テレオペは遅延で無理では?」→ 5分枠は見回り・水やり等の遅延耐性が高い操作に限定。精密作業はAI(PITCH-EXPERIENCE.md特注Q&A)
- 「エンタメで飢餓は救えないのでは?」→ 比喩ではなく損益計算書の話。体験課金が農地の運営費になる構造(同上)
