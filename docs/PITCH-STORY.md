# Agri Port ピッチ台本 — ストーリー版(田中さん軸)

## この版の設計思想

スタンダード版が「数字と構造」で押すのに対し、ストーリー版は**一人の顔(田中さん)を最初に立てて、全スライドを彼の物語として貫く**構成です。カッコつけた表現は削り、因果で話します:

```
田中さんの話(なぜ課題感を持ったか・どれほど深刻か)
 → 彼はひとりではない(数字)
 → なぜ止められなかったのか(構造の問題)
 → フィジカルAIの活路(労働と土地を切り離す)→「移住」から「ログイン」へ
 → 選ぶ・観る・受け取る → ライブデモ
 → 売れてから作る(「報われない」の逆転)
 → 田中さんのその後(ハッピーな結末+ソーシャルインパクト)
 → 田中さんは世界中にいる(課題の地球化)
 → 食で溢れる世界に変えられる(希望の地球化)
 → 国土の再稼働(野望) → チーム → クロージング
```

全14枚: `1 タイトル / 2 田中さん / 3 数字 / 4 なぜ未解決 / 5 活路 / 6 3ステップ / 7 ライブデモ / 8 売れてから作る / 9 田中さんのその後 / 10 世界中の田中さん / 11 食で溢れる世界(HOPE) / 12 国土再稼働 / 13 チーム / 14 クロージング`

**スタンダード版との違い**: Why Nowスライドを廃止(S5の「もう働いています」の一言に畳み込み)/ ミッション宣言はロードマップの見出しに統合 / ビジネス2枚→1枚 / 「農業OS」等の言い回しを平易化。

> ⚠️ **田中さんは統計から組み立てた架空のモデルケース(仮名)です。** スライドにもその旨を明記済み。もしチームの誰かに実の原体験(実家の田畑・祖父母の農地など)があるなら、S2をその実話に差し替えるのが最強です — 実話は捏造できない説得力を持ちます。

## 90秒版(9枚: 1→2→3→4→5→6→8→9→14)

> デモ(S7)は飛ばしてクロージングのQRに託す。デモを見せたい場なら S6 を飛ばして S7 を入れる。

**[S1] 0:00–0:07 — タイトル**
「Agri Port、アプリDE農業です。まず、ひとりの話をさせてください。」

**[S2] 0:07–0:22 — 田中さん**
「田中さん、72歳。新潟の谷あいで米を50年作ってきました。離農した近所の田も引き受けて、いまはこの谷で最後のひとりです。息子さんは東京、膝は限界。**田中さんが手を止めた日、この谷の田んぼは全部まとめて『荒廃農地』という統計に変わります。**(半拍)田中さんは仮名です。でも、数字は全部本物です。」

**[S3] 0:22–0:35 — 数字**
「日本の農家の70歳以上は55%。担い手は25年で57%減りました。そして作っても、手取りは小売価格の半分以下。価格は自分で決められない。豊作の年は1玉40円で畑に捨てる。**作る人が消えていく。作っても、報われない。** これが両輪で回っています。」

**[S4] 0:35–0:48 — なぜ止められなかったのか**
「なぜ誰も止められなかったのか。怠けていたからではありません。**産業のつくりの問題**です。関わる方法が移住しかない。労働が土地に縛られている。1軒では1,000万円の機械に投資できない。この3つが、ずっと塞いでいました。」

**[S5] 0:48–1:00 — 活路(キメ)**
「フィジカルAIは、この構造を初めて壊せます。**労働と土地を切り離せる**んです。ロボットが現地の身体になり、人はどこからでも繋がれる。収穫ロボもヒューマノイドも、もう商用で働いています。1台1,000万円も、1,000人で分ければ1人1万円。(半拍)**だから私たちは、農業を『移住』から『ログイン』に変えます。**」

**[S6] 1:00–1:08 — 3ステップ**
「アプリでポートを選ぶ。ロボットが育てる様子をライブで観る。野菜かお金か、リターンを受け取る。以上です。」

**[S8] 1:08–1:18 — 売れてから作る**
「そして『報われない』はこう逆転します。出資は前払購入なので、**作付け前に完売している**。手取りは48.5%から最大78.5%へ。廃棄は設計上ゼロ。」

**[S9] 1:18–1:26 — 田中さんのその後(感情のピーク)**
「田中さんの田んぼは、統計になりませんでした。田はポートになって地代が入り、田中さんは艦隊を見回るポートパートナーに。谷の新米は1,000人のオーナーに届き、秋には人が来ます。**お金は都市から地方へ。労働はロボットへ。野菜とデータと風景は、参加者へ。**」

**[S14] 1:26–1:33 — クロージング**
「田中さんは、世界中にいます。食の産業を守り切れば、世界の国土は**食で溢れる場所**に変えられる — まず日本の谷から。デモはこのQRから、いま触れます。Agri Portでした。」

## 150秒版(全14枚)

90秒版に以下を追加:

**[S7] — ライブデモ(S6の後)**
「実際の画面です。除草ロボの視点で、AIが雑草を検出して農薬ゼロで除去する瞬間を、出資者は誰でも見られます。フィジカルAIの視点と、あなたを繋ぐ — リモート参加型農業です。右端は選果場のヒューマノイド。畑から箱詰め・配送まで、全部この視点で観察できます。」

**[S10] — 世界中の田中さん(S9の後)**
「この課題は日本だけのものではありません。EUの農業経営者は61%が55歳以上。韓国は農家の51%が65歳以上。世界の耕作放棄地は約1億ha。日本はこの課題が最初に極まる場所 — **ここで解ければ、その答えはそのまま世界の田中さんに渡せます。**」

**[S11] — 食で溢れる世界(HOPE・新設)**
「ここからは、希望の話です。飢餓6.7億人、2050年に食料5割増し — 重い数字ですが、裏返せます。田中さんの谷で動いたこの静かな仕組みを、世界の眠る土地・約1億haに増やしていく。**食の産業を守り切ることは、それ自体が人類最大の飢餓対策なんです。** 世界の国土を、食で溢れる場所に変えられる。」

**[S12] — 国土の再稼働**
「私たちの野望はこれです。フルオート農業で、国土を再稼働する。2026年に1ポートから、2035年に1,000ポート、2040年には世界へ。担い手30万人の時代に、**関わる人1,000万人**の農業をつくります。」

**[S13] — チーム**
「空のJAL、土地と建築の大和ハウス、デジタルの富士通。ポートは空・土地・データが交差するインフラです。この3つの現場から集まった3人で構想しました。」

**[S14] — クロージング(150秒版)**
「地球で無人農業を解けない種は、火星でも耕せない — まず、田中さんの谷から。Agri Portでした。」

## 演出メモ

- **S2は静かに話す。** 数字もカタカナ語もなし。ここで会場の空気を変える
- S2→S3の橋: 「田中さんは仮名です。でも数字は全部本物です」→ S3の数字カウントアップへ
- **S5とS9が二大キメ**: S5「移住からログインに変えます」/ S9「統計になりませんでした」— どちらも1語ずつ
- S9のパワーライン(お金は都市から…)は画面の文字浮上と声を同期
- S14のQRは3秒黙って見せる
- 質疑の弾はスタンダード版 `PITCH.md` の想定Q&Aをそのまま使用
